## üÜò Troubleshooting

### File Won't Open
**Problem:** Double-clicking doesn't work
**Solution:** 
- Right-click ‚Üí "Open with" ‚Üí Select browser
- Make sure you have a modern browser installed

### Browser Compatibility
**Problem:** Features not working
**Solution:**
- Update your browser to latest version
- Try a different browser (Chrome recommended)

### Data Not Saving
**Problem:** Changes don't persist after closing
**Solution:**
- Check browser allows localStorage
- Don't use Private/Incognito mode
- Enable cookies and site data

### Mobile Display Issues
**Problem:** Layout looks wrong on phone
**Solution:**
- Rotate to landscape if needed
- Zoom out if necessary
- Use Chrome or Safari on mobile

### Setup Wizard Won't Start
**Problem:** Can't see setup button
**Solution:**
- Refresh the page (F5 or Cmd+R)
- Clear browser cache
- Try different browser
