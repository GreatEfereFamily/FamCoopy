## üì± Troubleshooting on Mobile:

### Can't Delete Repository?
- Make sure you're in Desktop Mode
- Must be in repository Settings (not account)
- Scroll ALL the way to bottom
- Look for red "Danger Zone"

### Can't Upload Files?
- Check file size (<100MB per file is fine)
- Make sure you're in the repository
- Use "Add file" ‚Üí "Upload files"
- Try one file at a time if multiple fail

### Desktop Mode Not Working?
- Close browser completely
- Reopen and try again
- Try different browser
- Clear cache if needed

### Files Not Showing?
- Wait a few seconds after upload
- Refresh the page
- Check "Commits" to see if upload succeeded
